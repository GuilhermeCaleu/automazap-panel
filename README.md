# AutomaZap Frontend PRO

Painel PRO em React + Vite para o backend PRO do AutomaZap.

- Login simples (localStorage)
- Tema dark/light
- Dashboard base
- Inbox consumindo `/api/messages`
- Contatos consumindo `/api/contacts`
- Regras consumindo `/api/rules`
- Configurações informativas

Configure `VITE_API_URL` no Render apontando para a URL do backend.
